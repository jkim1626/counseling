# Elite College Counseling Website

A professional Next.js website for tutoring and college counseling services, designed to attract students and parents interested in gaining admission to top universities.

## Features

### Navigation Tabs
The website includes four main sections accessible via tabs:

1. **About Us** - Introduction, mission statement, and key statistics
2. **Credentials** - Educational background, professional experience, and services
3. **Connect with Us** - Contact information and consultation request form
4. **Testimonies** - Success stories from students and parents

### Design Highlights
- Clean, professional blue/indigo color scheme
- Responsive design for mobile, tablet, and desktop
- Sticky navigation header
- Smooth animations and transitions
- Interactive form elements
- Comprehensive footer with quick links

## Getting Started

### Prerequisites
- Node.js 20.x or higher
- npm

### Installation

```bash
# Install dependencies
npm install
```

### Development

```bash
# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
# Create production build
npm run build

# Start production server
npm start
```

### Linting

```bash
# Run ESLint
npm run lint
```

## Technology Stack

- **Framework:** Next.js 16.0.1
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **React:** 19.2.0
